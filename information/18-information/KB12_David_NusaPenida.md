## CÓMO USAR ESTE ARCHIVO

Este archivo está diseñado para gestionar promociones y ofertas estacionales de forma **dinámica**, sin necesidad de modificar el prompt principal de David ni los otros KBs.

**Cuando se active una nueva oferta:**
1. Agregar la oferta en la sección "OFERTAS ACTIVAS" siguiendo la plantilla incluida abajo
2. Indicar fechas exactas de validez (inicio + fin)
3. Indicar a qué programas aplica y bajo qué condiciones
4. Subir el archivo actualizado al sistema → David la incorporará automáticamente

**Cuando una oferta expira:**
1. Mover la oferta de "OFERTAS ACTIVAS" a "HISTORIAL DE OFERTAS"
2. Anotar la fecha de cierre y resultados (opcional)
