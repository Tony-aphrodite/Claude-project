# KB-09 | PROGRAM DESCRIPTIONS — DPM DIVING NUSA PENIDA
## Text descriptions for WhatsApp (fallback when Meta catalog card is unavailable or insufficient)

---

## HOW TO USE THIS FILE

**Default behavior:** David's first move when a client asks about a program is to **fire the Meta catalog card** (see KB-01 for IDs). The card shows image + title + price + short description.

**When to use this KB-09 text instead:**
1. The card doesn't exist in the client's language (e.g., Scuba Diver has only EN card, no ES card)
2. The client asks for more detail than the card shows ("what exactly is included?")
3. The card fails to send (rare technical issue)
4. The client explicitly asks for text instead of a card

Keep messages natural — short, no bullet points, no markdown. 2-3 short WhatsApp messages max.
Always follow with a CTA (availability check or booking prompt).
